## Solutions for the Advent of Code 2019 in Ruby

https://adventofcode.com/2019/

My solutions for the Advent of Code 2019 written in Ruby

### Running
Run all, a specific day or part

`bin/run [<day> [<part>]]`

### Generate scaffold for new day

`bin/generate <day>` 

### Tests

Tests require `rspec`

`gem install rspec`

Run tests with 

`rspec`
